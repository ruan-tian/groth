四、动态经验值计算设计



学习经验



学习经验 =

基础时长经验

+ 专注度加成

+ 难度加成

+ 复盘加成





建议公式：



base = duration_minutes / 10



focus_bonus = focus_level * 2



difficulty_bonus = difficulty_level * 2



review_bonus = 是否填写收获或问题 ? 5 : 0



total = base + focus_bonus + difficulty_bonus + review_bonus





例子：



学习90分钟

专注度4

难度3

填写复盘



base = 9

focus_bonus = 8

difficulty_bonus = 6

review_bonus = 5



总经验 = 28









健身经验



健身经验 =

基础时长经验

+ 强度加成

+ 动作数量加成

+ 记录完整度加成





建议公式：



base = duration_minutes / 10



intensity_bonus = intensity_level * 3



exercise_bonus = 动作数量 * 2



complete_bonus = 是否填写训练感受 ? 5 : 0



total = base + intensity_bonus + exercise_bonus + complete_bonus









日记经验



日记经验 =

基础经验

+ 字数加成





建议公式：



base = 5



word_bonus = word_count / 100



total = base + word_bonus





设置上限：



日记经验每日最多 20 EXP





避免刷经验。







五、等级计算设计



不需要单独存等级，等级可以根据经验总和实时计算。



总经验 = growth_exp_log.exp_value 求和





等级规则：

建议使用公式：



level = floor(sqrt(total_exp / 100)) + 1





优点：



前期升级快

后期升级慢

适合长期成长









六、统计查询设计



今日学习时长



SELECT SUM(duration_minutes)

FROM study_record

WHERE date(created_at / 1000, 'unixepoch') = date('now');









本周健身次数



SELECT COUNT(*)

FROM fitness_record

WHERE created_at >= 本周开始时间;









年度学习趋势



SELECT 

  strftime('%Y-%m', datetime(created_at / 1000, 'unixepoch')) AS month,

  SUM(duration_minutes) AS total_minutes

FROM study_record

GROUP BY month;
