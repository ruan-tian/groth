## 二、任务分解（7个子任务）

### 任务 A：暗黑模式兼容 + 颜色系统迁移
**优先级：P0（阻塞其他UI工作）**

**改动范围：**
- `knowledge_workspace_page.dart` — 删除12个硬编码颜色常量，改用 `context.growthColors`

**方案：**
```dart
// 删除这些：
const _blue = Color(0xFF315BEF);
const _paper = Color(0xFFFFFFFF);
const _paperBorder = Color(0xFFE8ECF5);
const _ink = Color(0xFF172033);
const _muted = Color(0xFF667085);
const _softBlue = Color(0xFFEFF3FF);
// ... 等

// 替换为：
final colors = context.growthColors;
// _paper → colors.surface
// _ink → colors.textPrimary
// _muted → colors.textSecondary
// _blue → colors.study (或 colors.primary)
// _softBlue → colors.study.withValues(alpha: 0.08)
// _paperBorder → colors.border
```

**验收标准：** 切换暗黑模式后，知识卡片所有页面颜色正确、无白色刺眼区域

**依赖：任务 B（文件拆分）**

**数据库层（已存在，无需改动）：**

```sql
tiantian_qa_messages: id, session_id, role, content, sources_json, saved_as_card, created_at
```

**需要新增的 Repository 方法：**
```dart
// knowledge_v3_repository.dart 或新建 tiantian_chat_repository.dart
Future<int> createQaSession({required int spaceId, String? title, List<int>? materialIds});
Future<List<QaSession>> getSpaceQaSessions(int spaceId);
Future<List<QaMessage>> getSessionMessages(int sessionId);
Future<int> addQaMessage({required int sessionId, required String role, required String content, String? sourcesJson});
```

**AI Service 改造 — 流式输出：**

```dart
Stream<String> answerQuestionStream({
```

2. **流式输出**：使用 `StreamBuilder` 逐字显示，带打字机光标动画
3. **会话持久化**：每次进入默认继续上次会话，侧边栏可切换历史会话
4. **存为卡片**：AI 回答下方一键存为知识卡片
5. **追问**：在同一会话中继续对话

### 任务 E：资料提取 Prompt 优化
**依赖：无（可并行）**

**当前 Prompt 分析：**
- `cardSystemPrompt` 已经不错，但可以增强：
  - 缺少结构化标签体系
  - 难度分级标准不够明确
  - 没有要求 AI 生成"记忆口诀"或"助记法"

**优化方案：**
```
增强点：
1. 卡片类型细化：增加 "diagram"（图表理解）、"application"（应用场景）
2. 难度分级标准：1=基础记忆 2=理解运用 3=分析综合 4=易错陷阱
3. 增加 memoryHint 字段：要求 AI 生成记忆口诀/助记法
4. 增加 relatedCardTags 字段：便于后续关联推荐
5. sourceExcerpt 必须是原文精确摘录，不能改写
6. 覆盖率要求：信息密度高时宁多勿漏
```

**新增 JSON 字段：**
```json
{
  "question": "...",
  "answer": "...",
  "explanation": "...",
  "cardType": "recall|comparison|process|scenario|trap|cloze|choice|diagram|application",
  "importance": 1-4,
  "difficulty": 1-4,
  "sourceExcerpt": "原文摘录",
  "memoryHint": "记忆口诀/助记法（可选）",
  "tags": ["标签"],
  "relatedConcepts": ["相关概念"]
}
```

### 任务 F：复习页 AI 快速入口
**依赖：任务 D（对话系统）+ 任务 B（文件拆分）**

**在 `KnowledgeFlashReviewPage` 添加：**

1. **顶栏"问甜甜"按钮** — 针对当前卡片的知识点提问
2. **卡片详情页"一键询问AI"按钮** — 在卡片下方添加
3. **薄弱卡自动解释** — 复习完成后自动调用 `explainWeakCards`

**卡片内 AI 入口设计：**
```
┌─────────────────────────────────┐
│  Q: 泰勒展开和麦克劳林展开的    │
│     区别是什么？                 │
│                                 │
│  [显示答案]                      │
│                                 │
│  ┌─────────────────────────┐   │
│  │ A: 泰勒展开是在任意点   │   │  ← 翻转后显示
│  │ a 处展开...              │   │
│  │                          │   │
│  │ 💡 记忆口诀: 麦克=0点   │   │
│  │                          │   │
│  │ [🤖 问甜甜关于这个知识点] │   │  ← 一键询问
│  │ [📝 编辑] [🗑️ 归档]     │   │
│  └─────────────────────────┘   │
│                                 │
│  [😐 忘记] [😕 模糊] [😊 记得] │
└─────────────────────────────────┘
```

**点击"问甜甜"的流程：**
1. 自动构造问题：`"请详细解释：{card.question}"`
2. 自动关联该卡片的 `sourceExcerpt` 作为上下文
3. 打开 `TiantianChatSheet`，预填问题，自动发送
4. 回答中引用卡片来源资料
